Sprout collates certified EPD's for building materials from various sources for comparison. Each material has a set functional unit in either m2 or m3 either based on individual products, or an average from a number of supply chains. As these are done to ISO standards, there is no deviation possible from the function unit. These set thicknesses are labelled for each material, and should be considered when comparing materials.

The material's chosen for this database are all based on European production, and so are comparable between each other, but will not be representative of products from other continents.

The sourcing options provided are two set hypothetical distances that offer a generic comparison between sourcing products locally and internationally.

Glossary

EPD: An environmental product declaration (EPD) is a method of measuring the imapcts of a material through a particular supply chain

GWP: Global Warming Potential (GWP) is a measure of the impacts of various greenhouse gases trapped in the atmosphere

kg CO2e: Kilograms of carbon dioxide equivalent is a unit of measurement for the impacts to global warming from various sources, normalised to Carbon Dioxide

Sprout collates certified EPD's for building materials from various sources for comparison.

The materials are given in either m2 for Flooring and Cladding, m3 for Structural, and a mix for Other.

There are options to choose generic distances for Local and International sourcing of the materials to give an insight into transportation impacts.

This is an advisory tool that has a number of assumptions, and should be used as only a generic guideline.
